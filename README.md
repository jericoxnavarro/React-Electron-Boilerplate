## Getting Started :green_heart:

This is how to run the files.

### Installation :alien:

1. Clone the repo
   ```sh
   git clone https://github.com/jericoxnavarro/React-Electron-Boilerplate your-project-name
   ```
2. cd to project
   ```sh
   cd your-project-name
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Start Project
   ```sh
   yarn start
   ```
5. Build Project
   ```sh
   yarn build
   ```
